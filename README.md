## Production Projects

### DNA Analysis LIMS
Laboratory information management system for forensic DNA sample tracking.

[![Live](https://img.shields.io/badge/Live-lims.jagdevops.co.za-16a766?style=flat-square)](https://lims.jagdevops.co.za)
[![Repo](https://img.shields.io/badge/Repo-JAG--LABSCIENTIFIC--DNA-181717?style=flat-square&logo=github)](https://github.com/GABRIELS562/JAG-LABSCIENTIFIC-DNA)

- **Stack:** React 18, Node.js, PostgreSQL, K3s, GitHub Actions
- **CI/CD:** 6 workflows, 3 environments (production/develop/test)
- **Features:** ArgoCD GitOps, HPA autoscaling, Vault secrets

### Cloud-Native eShop
14-service microservices e-commerce platform with polyglot architecture.

[![Live](https://img.shields.io/badge/Live-eshop.jagdevops.co.za-16a766?style=flat-square)](https://eshop.jagdevops.co.za)
[![Repo](https://img.shields.io/badge/Repo-eshop--platform--infra-181717?style=flat-square&logo=github)](https://github.com/GABRIELS562/eshop-platform-infra)

- **Stack:** Go, C#, Python, Node.js, Java (gRPC communication)
- **Infrastructure:** Redis, RabbitMQ, PostgreSQL, OpenTelemetry
- **GitOps:** 17 ArgoCD applications with auto-sync

### Forensic Evidence Collector
Compliance automation with tamper-evident blockchain-style audit trails.

[![Live](https://img.shields.io/badge/Live-dashboards.jagdevops.co.za-16a766?style=flat-square)](https://dashboards.jagdevops.co.za)
[![Repo](https://img.shields.io/badge/Repo-forensic--evidence--collector-181717?style=flat-square&logo=github)](https://github.com/GABRIELS562/forensic-evidence-collector)

- **Stack:** Python 3.11, SQLite, Docker, Prometheus
- **Integrity:** SHA-256 hash chains, 8,500+ evidence blocks
- **Metrics:** Real-time Prometheus endpoints
