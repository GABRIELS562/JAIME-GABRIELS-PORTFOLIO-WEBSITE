# Jaime Gabriels Portfolio

Professional portfolio website for Jaime Gabriels - Passionate DevOps Engineer, built with React and modern design principles.

## ✨ Features
- **Typewriter Animation**: Dynamic greeting with "Hi 👋 I'm" and animated title
- **3D Card Effects**: Interactive cards with lean-back hover animations
- **Theme Toggle**: Dark/Light mode with smooth transitions
- **Responsive Design**: Fully responsive across all devices
- **Modern UI**: Clean, professional design with gradient effects
- **Technology Showcase**: Interactive technology icons with hover effects
- **Sticky Navigation**: Floating navigation that follows scroll
- **DevOps Background**: Dynamic background images that change with theme

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/jaime-gabriels-portfolio.git
   cd jaime-gabriels-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment

### Option 1: Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Option 2: Netlify
```bash
npm run build
# Then drag and drop the 'build' folder to Netlify
```

### Option 3: GitHub Pages
```bash
npm install --save-dev gh-pages
npm run build
npx gh-pages -d build
```
